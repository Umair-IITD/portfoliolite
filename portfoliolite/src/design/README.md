# Design Reference Files

These files are copied from the Figma Make prototype and serve as the
visual specification for the PortfolioLite app.

## Purpose
- Visual reference while building screens in `src/screens/`
- Source of design tokens (colors, spacing, fonts)
- Component structure reference for the scaffold prompt

## Important
These files are NEVER imported into the app.
Do not add them to any import or index file.
The real app screens live in `src/screens/` (built in Week 2).

## Design Tokens (copy these into your actual screens)
- Background: #0A0F1E
- Card: #111827
- Accent: #3B82F6 or #00D4B4
- Gold (₹ values): #F5A623
- Green (gains): #22C55E
- Red (losses): #EF4444
- Text primary: #F1F5F9
- Text secondary: #94A3B8
- Text muted: #64748B
- Card radius: 16px
